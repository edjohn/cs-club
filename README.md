## Week 1-2
**Linux & The Shell**
- *Meeting Date: 09/18/2020*
1. Follow the instructions [here](https://docs.google.com/document/d/1ysMMv5nPhDEp0-l90vsOX99Jo3HtsrAspkxrqPn4jbE/) to create a Linux VM
